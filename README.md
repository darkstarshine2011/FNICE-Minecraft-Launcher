# FNICE Launcher

A sleek, modern Minecraft launcher built with HTML, CSS, JavaScript, and Python (`pywebview`). Features a highly customized UI with dynamic particle backgrounds, smooth gradients, and gold-accented themes.

یک لانچر شیک و مدرن برای ماینکرفت که با استفاده از HTML، CSS، جاوااسکریپت و پایتون (`pywebview`) ساخته شده است. این لانچر دارای یک رابط کاربری کاملاً شخصی‌سازی شده به همراه پس‌زمینه متحرک ذرات، گرادینت‌های روان و تم با جزئیات طلایی است.

---

## 🚀 Features | قابلیت‌ها

* **Dynamic UI:** Elegant pure blue to black gradient background with gold styling.
* **Performance:** Lightweight UI rendering via `pywebview` separate from backend operations.
* **Smart Storage:** Automatically remembers and loads the last used username (`localStorage`).
* **Mod Loader Support:** Built-in compatibility structure for Vanilla, Fabric, Forge, and NeoForge.
* **رابط کاربری پویا:** پس‌زمینه شیک با گرادینت آبی به مشکی و استایل دکمه‌ها و المان‌های طلایی.
* **کارایی بالا:** رندینگ سبک رابط کاربری از طریق `pywebview` و مجزا از عملیات بک‌اند.
* **ذخیره‌سازی هوشمند:** به یاد سپردن و لود خودکار آخرین نام کاربری استفاده شده (`localStorage`).
* **پشتیبانی از مادلودرها:** ساختار سازگار داخلی برای نسخه‌های Vanilla، Fabric، Forge و NeoForge.

---

## 🛠️ Built With | تکنولوژی‌های استفاده شده

* **Frontend:** HTML5, CSS3 (Custom variables & animations), JavaScript (ES6)
* **Backend Interface:** Python (`pywebview` API)
* **Icons:** FontAwesome
